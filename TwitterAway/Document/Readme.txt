TwitterAway
---


【主な機能】
TwitterAwayはMicrosoft Windows Mobile 5.0 Pocket PC上で動作するTwitterクライア
ントです。


【必要環境】
Microsoft Windows Mobile 5.0 Pocket PC
.NET Compact Framework 1.0 SP3

.NET Compact Frameworkについては
http://www.microsoft.com/japan/msdn/vstudio/device/netcf/
を参照してください。


【インストール】
ZIPファイルを解凍し、TwitterAway.exe・MiscPocketCompactLibrary.dllを適当なフォ
ルダに置いてください。


【アップデート】
ZIPファイルを解凍し、TwitterAway.exe・MiscPocketCompactLibrary.dllを適当なフォ
ルダに置いてください。


【アンインストール】
TwitterAway.exe・MiscPocketCompactLibrary.dll、同じフォルダにあるSetting.xmlを
削除してください。
TwitterAwayはレジストリを使用しません。


【使い方】
・TwitterAwayの使い方
Updateボタンを押し、Friendsのリストを取得します。
テキストボックスにメッセージを入力し、Updateボタンを押すと、自分のステータスを
更新します。

・TwitterAwayの設定
トップ画面から[メニュー]->[TwitterAway設定]を選択するとTwitterAwayの設定を行う
ことができます。
[ユーザー名]はTwitterのユーザー名を設定します。
[パスワード]はTwitterのパスワードを設定します。
